# HACKATHON.EXE - MONOLITH HACKATHON

## Overview
Monolith is a 5-week sprint to build a mobile app for the Solana dApp Store.

## Prizes
- **$10,000**: 10 Winners
- **$5,000**: 5 Honorable Mentions
- **$10,000**: SKR Bonus

- **Integration**: Must integrate Solana Mobile Stack (SMS) and Mobile Wallet Adapter (MWA).
- **Design**: Built for mobile from the ground up (no PWA wrappers).
- **Meaning**: Must interact meaningfully with the Solana network.

## Evaluation Criteria (25% each)
1. **Stickiness & PMF**: Resonates with Seeker community, drives engagement.
2. **User Experience**: Intuitive, polished, enjoyable.
3. **Innovation / X-factor**: Novel and creative idea.
4. **Presentation & Demo**: Clear communication and effective demo.
